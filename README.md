# Marketing Resource Hub

一个全面的营销资源整合平台，为营销专业学生和从业者提供学习和发展资源。

## 功能特点

- 📚 营销资料库：全面的营销学习资源
- 📝 营销资讯：最新营销趋势和播客资源
- 📊 数据分析学习：系统的数据分析课程
- 💼 求职招聘：营销行业求职资源
- 🤖 AI助手：智能营销咨询助手
- 👥 营销社区：专业营销交流平台

## 本地运行

1. 克隆仓库
```bash
git clone [repository-url]
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
streamlit run app.py
```
